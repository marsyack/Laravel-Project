# Laravel Project

Project Web Lanjut menggunakan Laravel.

## Deskripsi
Project ini dibuat untuk memenuhi tugas praktikum Web Lanjut.

## Teknologi
- Laravel
- PHP
- MySQL

## Cara Menjalankan
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` ke `.env`
4. Jalankan `php artisan key:generate`
5. Jalankan `php artisan serve`
